# Unified Embedded Filter Optimizer - TODO List

이 리스트는 향후 UI 완성도와 기능 정교화를 위해 해결해야 할 과제들을 정리한 것입니다.

## UI/UX 개선
- [ ] **데이터 규모(n) 레이아웃 대응:** 입력값이 매우 클 때(예: 100B 이상) 보라색 단위 표시가 입력창의 숫자와 겹쳐 보이는 현상 해결.
- [ ] **FPR 입력 제한:** Target FPR(p) 입력 시 소수점 오차나 너무 긴 소수점 입력을 방지하기 위해, 사용자 입력값을 정수 단위의 퍼센트(%) 형태로 제한하거나 보정하는 로직 추가.

## 기능 확장
- [ ] **개별 필터 박스 내 FPR 표시:** 사용자가 하단에서 수동(Manual)으로 파라미터(k, b, f 등)를 조정함에 따라 실시간으로 변화하는 해당 필터만의 실제 FPR 값을 카드 내부에 명시하여 성능 변화를 직관적으로 확인하게 함.
